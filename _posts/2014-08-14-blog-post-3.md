现根据课题需要招聘<b><u>硕士或博士研究生</u></b>若干，详细情况见链接内。  
1. 按照中国药科大学的招生要求取得硕士或博士研究生入学资格；
2. 具有良好的中英文写作以及沟通能力，可协助完成科学基金和学术论文的写作；
3. 热爱科研、责任心强、学习能力强、拥有良好的团队精神与合作意识；
4. 具有一定编程、计算机、以及深度学习基础者优先考虑；
5. 具有一定细胞生物学实验，分子生物学实验，生物物理学实验、小鼠实验、高通量测序技术基础者优先考虑；

岗位职责
------
1. 作为团队主要成员具有开展研究课题的能力；
2. 协助指导实习生和技术人员开展工作；
3. 积极参与实验室安排的其他工作和团队工作。

资格要求
------
1. 按照中国药科大学的招生要求取得硕士或博士研究生入学资格
2. 有意致力于“人工智能驱动的药物研发和功能验证”相关研究
3. 具有一定计算机和深度学习基础者优先考虑
4. 具有一定分子、细胞、及小鼠实验基础者优先考虑

岗位待遇
------
1. 学校经费+课题组劳务（面议）；
2. 全力支持申报各类国家和省级的基金项目
3. 课题组前期数据丰富，提供充足科研经费支持，优秀研究生可快速产出，科研方面取得突出成果者，享受学校相应的奖励政策外，课题组将给予额外奖励。
4. 课题组与清华大学、北京大学、哈佛医学院等建有长期合作关系，可推荐前往交流学习。

申请方式
------
请将请将以下材料发送至邮箱：
1. 个人简历简短的个人陈述；
2. 申请原因及研究规划；
3. 成绩单
4. 发表论文清单（如有）

对于合适的应聘者，我们将尽快联系并安排面试。
